# dio_ransomware_e_keylogger
Prática do Lab Project "Simulando um Malware de Captura de Dados Simples em Python e Aprendendo a se Proteger" da plataforma de cursos Dio.me

# Descrição do Projeto

O projeto consiste em três scripts Python que simulam o comportamento de malwares sendo um deles um ransomware, um keylogger e um keylogger que envia os dados coletados da vitima para o e-mail do atacante.
Além disso, será discutido nesse arquivo README.md formas de e prevenir de ataques que utilizam esses malwares.
Os scripts criados para este projéto foram feitos apenas para fins educacionais e não devem ser utilizados para fins maliciosos.

# Como se proteger de um ataque de um malware

- Manter os antivirus e firewalls sempre atualizados;
- Monitorar o comportamento dos aplicativos;
- Ter cuidado ao acessar links e abrir arquivos suspeitos;
- Utilizar ambientes isolados para testes para a execução de arquivos suspeitos.
